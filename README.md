# 注意力计算器.Skill

这是“注意力计算器”的 Skill 版本，用于远程调用 zhuyidao.com/zhuyidao.net上的“注意力计算器”服务，生成关于 `e`、`π` 、三角函数等29种类型的不等式的积分证明。

## 使用方式

安装“注意力计算器.skill" 后，可以直接提问，例如：

```text
用注意力计算器证明 pi > 3
```

```text
用注意力计算器生成 e < 3 的积分证明
```
